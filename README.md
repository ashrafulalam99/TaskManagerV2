# Task Management API

A task management backend built with Node.js, Express and MYSQL.

## Features

- JWT authentication and role-based authorization
- Validate user input and error-handling by logging
- Create, read, update, and delete own tasks as user
- Read tasks of all users, delete user as admin
- RESTful API endpoints
- Database queries

## Tech Stack

- Node.js
- Express.js
- MYSQL

## Getting Started

### Prerequisites
- Node.js installed on your system
- MYSQL installed

### Installation

1. Clone the repository

- git clone: https://github.com/ashrafulalam99/taskmanagerV2.git
- cd taskmanagerV2

2.Install dependencies and set up environment

- npm install
- Create Database following schema

3. Start the server or for development

- npm start
- npm run dev

Use API testing tools like Postman or Thunder Client to interact with the endpoints. Must consider jwt authentication while testing.
